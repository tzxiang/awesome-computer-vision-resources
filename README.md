# Dataset

##  - Object Detection Dataset  
- **DOTA Aerial Detection dataset**  
Paper: DOTA：A Large-scale Dataset for Object Detection in Aerial Images, CVPR2018  [[Dataset]](http://captain.whu.edu.cn/DOTAweb/)

- **VisDrone2018/2019**  
Vision Meets Drones: A Challenge. It includes 4 tasks:   
(1) Task 1: object detection in images challenge. The task aims to detect objects of predefined categories (e.g., cars and pedestrians) from individual images taken from drones.   
(2) Task 2: object detection in videos challenge. The task is similar to Task 1, except that objects are required to be detected from videos.  
(3) Task 3: single-object tracking challenge. The task aims to estimate the state of a target, indicated in the first frame, in the subsequent video frames.  
(4) Task 4: multi-object tracking challenge. The task aims to recover the trajectories of objects in each video frame.  
[[Dataset]](http://www.aiskyeye.com/) [[PDF]](http://www.aiskyeye.com/upfile/Vision_Meets_Drones_A_Challenge.pdf)

- **UCAS-AOD**  
Dataset of Object Detection in Aerial Images, including cars and aircraft. 中科大模式识别实验室标注的，只包含两类目标：汽车，飞机，以及背景负样本。
Paper: Orientation Robust Object Detection in Aerial Images Using Deep Convolutional Neural Network, ICIP, 2015.

- **RSOD-Dataset**
An open dataset for object detection in remote sensing images, including aircraft, playground, overpass and oiltank. It is collected by Zhifeng Xiao, LIESMARS, WHU.
Paper: Accurate Object Localization in Remote Sensing Images Based on Convolutional Neural Networks, TGRS2016
[[Dataset]](https://github.com/RSIA-LIESMARS-WHU/RSOD-Dataset-)

- **NWPU VHR-10**  
西北工业大学标注的航天遥感目标检测数据集，共有800张图像，其中包含目标的650张，背景图像150张，目标包括：飞机、舰船、油罐、棒球场、网球场、篮球场、田径场、港口、桥梁、车辆10个类别。开放下载，大概73M. 
Paper: A survey on object detection in optical remote sensing images, ISPRSJ2016
[[Dataset]](http://jiong.tea.ac.cn/people/JunweiHan/NWPUVHR10dataset.html)

- **INRIA aerial image dataset**
Inria Aerial Image Labeling Dataset 数据集是一个城市建筑物检测的遥感图像数据集，标记只有建筑/非建筑两种，且是像素级别，用于语义分割。该数据集于 2017 年由 Inria 发布，相关论文为 Can Semantic Labeling Methods Generalize to Any City? The Inria Aerial Image Labeling Benchmark。[[Dataset]](https://project.inria.fr/aerialimagelabeling/)

- **Flying Object Detection dataset**
To detect flying objects such as UAVs and aircrafts.
Reference paper: Detecting Flying Objects using a Single Moving Camera, CVPR2015, PAMI2017
[[Dateset&Codes]](https://www.epfl.ch/labs/cvlab/research/uav/research-unmanned-detection/)

- **Nanonets**  
A company to create custom Deep Learning models to detect a variety of objects in images captured by drones with minimal data & limited knowledge of machine learning. [[Homepage]](https://nanonets.com/drone/)
